
# 🥁 Drum Kit Web Application

The Drum Kit is an interactive web application that allows users to play different drum sounds using their keyboard or by clicking on the drum buttons displayed on the screen. Each button represents a different drum instrument and produces a unique sound when activated.

This project demonstrates how JavaScript can be used to handle user interactions such as keyboard presses and mouse clicks to create an engaging and interactive musical experience.

---

##  Features

- Interactive drum buttons
- Play drum sounds using keyboard keys
- Click-based sound interaction
- Simple and responsive design
- Real-time audio feedback
- Smooth and engaging user experience

---

##  Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript**
- **Google Fonts**

---

##  How It Works

The drum kit contains several buttons, each linked to a specific keyboard key.

When a user:
- **Presses a key on the keyboard**, or
- **Clicks on a drum button**

The corresponding drum sound is played instantly.

Example key controls:

| Key | Drum Sound |
|----|----|
| W | Drum Sound |
| A | Drum Sound |
| S | Drum Sound |
| D | Drum Sound |
| J | Drum Sound |
| K | Drum Sound |
| L | Drum Sound |
